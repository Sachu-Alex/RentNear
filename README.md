# 📦 RentNear - Flutter Rental Application UI

A modern **Flutter-based mobile UI** for a rental platform that allows users to discover and rent nearby items like weighing machines, tools, and other equipment. Designed with a focus on **clean UI, smooth UX, and scalable architecture**.

---

## ✨ Features

* 🔐 **Authentication UI**

  * Login screen
  * Registration screen
  * Form validation ready
  * Social login UI (Google / Apple)

* 🏠 **Home Screen**

  * Search bar for quick item discovery
  * Category-based browsing
  * Featured items carousel
  * Modern item cards with pricing and ratings

* 📱 **User Experience**

  * Clean and minimal interface
  * Smooth navigation
  * Mobile-first responsive layout
  * Reusable Flutter widgets

---

## 🧠 Concept

**RentNear** enables users to **find and rent items nearby**, promoting:

* Cost-effective usage
* Access over ownership
* Easy discovery of local rentable products

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/rentnear.git
```

### 2. Navigate to project folder

```bash
cd rentnear
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Run the app

```bash
flutter run
```
